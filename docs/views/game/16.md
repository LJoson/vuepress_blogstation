# 引言
C++实现基于SFML库的十六个经典游戏小项目。

## 环境
- vs2017
- SFML64位，[安装配置](https://www.sfml-dev.org/tutorials/2.5/start-vc.php)
如果曾经配置过vs+opencv开发环境，会发现很熟悉。基本是包含目录-其他库目录-依赖项，需要注意的是debug跟release版本的依赖名字有点不同。
